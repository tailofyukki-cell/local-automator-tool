# Local Automator

Microsoft Power AutomateのようにGUIでフローを作成・実行できる、完全ローカル動作のWindows専用自動化ツールです。

## 特徴

- **完全オフライン動作**: インターネット接続は一切不要です。外部APIやCDNへのアクセスもありません。
- **ポータブル**: 1つの.exeファイルのみで動作し、インストールは不要です。好きなフォルダに置いてすぐに使えます。
- **プライバシー重視**: テレメトリや自動アップデート機能など、外部への通信機能は一切ありません。フローやログはすべてローカルに保存されます。

## 機能

- **GUIフロービルダー**: ドラッグ＆ドロップで直感的に自動化フローを構築できます。
- **多彩なアクション**: ファイル操作、コマンド実行、変数処理、条件分岐など、基本的な自動化に必要なアクションを提供します。
- **実行エンジン**: 作成したフローをGUI上から実行し、各ステップの成功・失敗をリアルタイムで確認できます。
- **ログ機能**: フローの実行結果はすべてログファイルとして保存され、いつでも確認できます。

## 成果物

1.  **ソースコード一式**: このリポジトリに含まれるすべてのファイルです。
2.  **単体.exeファイル**: GitHub Actionsの[Artifacts](https://github.com/tailofyukki-cell/local-automator-tool/actions)からダウンロードできます。
3.  **サンプルフロー**: `samples/`フォルダに、ツールの使い方を示すいくつかのサンプル（.jsonファイル）が含まれています。

## ビルド手順

本プロジェクトは、リポジトリにソースコードがプッシュされると、GitHub Actionsによって自動的にWindows向けの.exeファイルがビルドされます。

手動でビルドを実行したい場合は、以下の手順に従ってください。

1.  **リポジトリをクローンします。**
    ```bash
    git clone https://github.com/tailofyukki-cell/local-automator-tool.git
    cd local-automator-tool
    ```

2.  **Python環境をセットアップします。** (Python 3.11推奨)

3.  **必要なライブラリをインストールします。**
    ```bash
    pip install -r requirements.txt
    ```

4.  **PyInstallerでビルドを実行します。**
    ```bash
    pyinstaller build.spec --onefile
    ```

5.  ビルドが成功すると、`dist`フォルダ内に`LocalAutomator.exe`が生成されます。

| :--- | :--- | :--- |
| ファイル操作 | フォルダ作成 | 指定したパスにフォルダを作成します。 |
| | フォルダ削除 | 指定したフォルダを中身ごと削除します。 |
| | ファイルコピー | ファイルを指定した場所にコピーします。 |
| | ファイル移動 | ファイルを指定した場所に移動します。 |
| | ファイル削除 | 指定したファイルを削除します。 |
| | ファイルリネーム | ファイルまたはフォルダの名前を変更します。 |
| | ファイル一覧取得 | フォルダ内のファイル一覧を変数に保存します。 |
| | テキスト読み込み | テキストファイルの内容を変数に保存します。 |
| | テキスト上書き | テキストファイルに内容を上書き保存します。 |
| | テキスト追記 | テキストファイルに内容を追記します。 |
| コマンド実行 | コマンド実行 | 任意のコマンド（.exe, .bat, PowerShell）を実行します。 |
| 変数処理 | 変数を設定 | 変数を作成または更新します。 |
| | 文字列結合 | 複数の文字列を結合して変数に保存します。 |
| | 文字列置換 | 文字列内の特定のテキストを置換します。 |
| | 日付取得 | 現在の日付/時刻を指定のフォーマットで取得します。 |
| | 数値計算 | 数式を評価して計算結果を変数に保存します。 |
| 条件分岐 | IF 条件分岐 | 条件を評価し、結果に応じて後続のアクションをスキップします。 |
| | END IF | IF条件分岐の終端を示します。 |

## サンプルフロー

- **`samples/sample_backup.json`**: 指定したファイルを日付名のフォルダにバックアップするフローです。
- **`samples/sample_conditional_process.json`**: ファイルの存在をチェックし、存在する場合のみ特定の処理を行う条件分岐を含んだフローです。
- **`samples/sample_variables.json`**: 変数を使って動的にファイル名やパスを生成し、レポートファイルを作成するフローです。
